## organic_waste_desert_relay_simulation.py — グラフの読み方

このファイルは `organic_waste_desert_relay_simulation.py` が生成する3つのグラフについて、
日本語で説明したものです。

**重要：すべてのグラフは概念実証モデルの出力です。**
パラメータ値・数値・軌跡はすべて仮説的なものであり、実測値ではありません。
科学的・衛生的・生態学的・農業的・工学的な独立した検証なしに、
実際の政策・農業・工学上の意思決定に使用することはできません。

## グラフ 1：`organic_waste_desert_relay_main.png`

**メインシナリオ比較（4×2パネル）**

### 横軸
- 横軸はすべて「西暦年」（2026年〜2075年、50年間）

### 縦軸
- 各パネルに異なる概念的指標（すべて0〜1の正規化インデックス、または相対単位）

### 4本の線の意味

| 線の色・スタイル | シナリオ | 意味 |
|---|---|---|
| グレー（実線） | 1. Baseline Desert | 何も介入しない基準シナリオ。緩やかに劣化し続ける |
| 赤（破線） | 2. Organic Waste Only | 生ごみを最小限の処理で直接投棄。高リスク・低効果 |
| 青（実線） | 3. HRS + DGS | 完全処理済み有機物をHRSで腐葉土化し、砂漠緑地化に活用 |
| 緑（太実線） | 4. HRS + DGS + Food Relay | 上記に加え、段階的食料遷移リレーを組み合わせた完全システム |

### 各パネルの読み取り方

**Soil Carbon Index（土壌炭素指標）**
- ベースラインは微弱な劣化。
- シナリオ3・4で腐葉土プールが蓄積されると、年々土壌炭素が改善。
- 50年後のシナリオ3は約0.33、シナリオ4は約0.41（いずれも仮説値）。

**Water Retention Index（保水性指標）**
- 腐葉土の有機物が土壌の保水力を改善する仮説を反映。
- シナリオ3・4では50年後に0.47〜0.50に到達。
- DGS起動条件（保水指標0.12以上）を満たすには本モデルで約19年を要する。

**Microbial Diversity Index（微生物多様性指標）**
- **シナリオ2（赤）が0まで低下することに注目**：
  未処理の有機廃棄物を投棄すると病原菌汚染で土壌微生物が崩壊する可能性を示す。
  これはゼロリスク処理が不可能であるという前提を強調している。
- シナリオ4（緑）では食料リレーのフェーズ2（マメ科）以降、根圏微生物の改善が加速。

**Vegetation Cover Index（植生被覆指標）**
- DGSが起動する2045年ごろから急激に上昇（シナリオ3・4）。
- シナリオ4ではさらに食料リレーのPhase 1（イモ類）が地表被覆を加速。

**Food Production Potential（食料生産ポテンシャル）**
- シナリオ1・2・3ではほぼゼロのまま推移。
- シナリオ4では食料リレー起動（2050年）後にPhase 1（イモ類）で緩やかに上昇。
  50年後に約0.030（仮説値）。Phase 2・3でさらなる向上が期待されるが、50年内は一部のみ。

**Carbon Fixation Potential（炭素固定ポテンシャル）**
- 植生が増加するとともに炭素固定能も上昇。
- シナリオ4では生物多様性フェーズ（Phase 3）のハーブ・混合植生が固定能を後押し。

**Residual Biosecurity Risk（残留バイオセキュリティリスク）**
- **重要：このモデルではゼロリスクは存在しない。**
- シナリオ1（未介入）はリスクゼロだが、それは「有機物を何も投入していない」ため。
- シナリオ2（最低限処理）は約0.15のリスクに収束。
- シナリオ3・4（HRS完全処理）でも約0.089に収束する。これが処理後の残留リスクのフロアを示す。

**Cumulative Humus Pool（腐葉土プール）**
- HRS処理された有機物が蓄積する中間プール。
- 年間の新規投入と自然的ターンオーバー（8%/年）のバランスで定常状態に収束する仮説。

### 注意すべき限界

- DGSの起動が2045年と遅く、食料リレーは2050年にようやく始まる。
  これは本モデルのパラメータ設定（保水閾値・DGS成長率）に依存しており、
  現実の気候・土壌・水資源条件によって大きく変わる。
- 食料生産ポテンシャルが低いのは、50年以内にフェーズ3まで到達しきれないため。
  長期シミュレーションではより大きな改善が見込まれる（ただし未検証）。
- すべての数値は正規化された仮説的指標であり、実際のトン数・面積・温度とは対応しない。

## グラフ 2：`organic_waste_preprocessing_pipeline.png`

**前処理パイプライン分析（2パネル）**

### 左パネル：廃棄物種別生成プロファイル

**横軸**
- 有機廃棄物の6種類（食品ロス・台所ごみ・落ち葉・枝・紙/段ボール・農業残渣）

**縦軸**
- 相対的な年間生成量（仮説的な相対単位。実際のトン数ではない）

**読み取れること**
- 食品ロス（3.5）と台所ごみ（2.8）が最大の潜在的有機物資源。
- 農業残渣（2.5）も大きなポテンシャルを持つが、農薬・除草剤残留リスクが課題。
- これらすべてが適切に処理・循環された場合、相当量の腐葉土原料になり得る（仮説）。

**注意すべき限界**
- 各数値は地域・季節・食文化によって大きく異なる。
- 分離収集率は社会的・行動的条件に強く依存する。

### 右パネル：処理レベル別 有効収量 vs 残留リスク

**横軸**
- 3つの処理レベル：無処理 / 最低限処理 / HRS完全処理

**縦軸（青：左軸）**
- 前処理後の有効収量（仮説的相対単位）

**縦軸（赤：右軸）**
- 残留バイオセキュリティリスク（仮説的指標）

**読み取れること**
- 無処理では分離収集率が低く（10%）、殺菌ゼロのため有効収量は低いが、
  リスクは0.25と最高（投入しない場合のリスクとは別の問題）。
- HRS完全処理では有効収量が最大になる一方、リスクは0.089まで低下。
- **しかし、0にはならない：** 完全処理後も0.089のリスクが残る（フロア値）。
  これが「病原菌リスクゼロは達成不可能」というモデルの中核的主張。

**注意すべき限界**
- リスクフロア（0.089）は仮定値。実際の残留リスクは処理方法・材料・規制基準に依存。
- 「HRS完全処理」の定義もここでは仮説的であり、
  実際の処理プロトコルの設計・検証が別途必要。

## グラフ 3：`organic_waste_food_transition_relay.png`

**段階的食料遷移リレーのフェーズ比較（2×2パネル）**

### 横軸
- 西暦年（2026〜2075年）

### 縦軸
- 各パネルに対応する仮説的指標（0〜1正規化）

### 2本の線の意味

| 線の色・スタイル | 意味 |
|---|---|
| 青破線 | シナリオ3：HRS + DGS のみ（食料リレーなし） |
| 各色実線 | シナリオ4：HRS + DGS + 食料リレー（完全システム） |

### フェーズ着色の意味

| 色 | フェーズ | 時期（本モデルの仮説）| 主な作物 |
|---|---|---|---|
| 薄黄色 | Phase 1：Tuber（イモ類） | 2050〜2055年頃 | ジャガイモ・サツマイモなど |
| 薄青色 | Phase 2：Legume（マメ科） | 2055〜2065年頃 | 豆類・窒素固定作物 |
| 薄緑色 | Phase 3：Herb/Mixed（ハーブ・混合） | 2065年〜 | ハーブ・薬草・花粉媒介植物など |

**注意：フェーズ境界は近似的なもので、実際の起動タイミングはDGS起動条件（保水指標の閾値）に依存する。**

### 各パネルの読み取り方

**Vegetation Cover Index（植生被覆指標）**
- シナリオ3・4ともに2045年ごろからDGSが起動し、植生が増加し始める。
- シナリオ4はPhase 1（イモ類）の地表被覆加速効果でさらに速く成長。
- 50年後のシナリオ4：約0.486、シナリオ3：約0.423（いずれも仮説値）。

**Food Production Potential（食料生産ポテンシャル）**
- シナリオ3ではほぼ0のまま（食料リレーなし）。
- シナリオ4ではPhase 1開始（2050年）から緩やかに上昇し、約0.030に到達。
- Phase 2・3への移行でさらなる上昇が期待されるが、本シミュレーションの50年枠では部分的。

**Microbial Diversity Index（微生物多様性指標）**
- シナリオ3と4の差が最も大きいパネル。
- Phase 2（マメ科・窒素固定）での根圏微生物の多様化が大きな寄与。
- Phase 3（ハーブ・混合植生）ではさらに多様化が加速。
- 50年後：シナリオ4 ≈ 0.543、シナリオ3 ≈ 0.304（仮説値）。

**Carbon Fixation Potential（炭素固定ポテンシャル）**
- Phase 3の多様植生が炭素固定ポテンシャルを最も大きく向上させる。
- ただし50年後でもシナリオ4で約0.149と限定的（砂漠からの出発点を考えると相当な改善だが、
  仮説値であり惑星規模の気候へのインパクトは独立した定量化が必要）。

### 注意すべき限界

- フェーズ境界（薄色の帯）は本モデルの仮説的な近似であり、
  実際の農業遷移タイミングは気候・土壌・水・種選択・病害虫圧によって大きく異なる。
- Phase 3は本シミュレーションでは2065年以降の約10年間しか描写できていない。
  より長期のシミュレーション（100年以上）での検証が望まれる。
- 食料生産ポテンシャルは「多様性・栄養バランス」を含まない単純指標であり、
  現実の食料安全保障評価には適さない。

## まとめ：3つのグラフから読み取れる構造的メッセージ

1. **無介入でも、悪い介入（未処理ごみの投棄）は選択肢ではない。**
   シナリオ2（最低限処理）は微生物多様性が崩壊し、リスクは残留する。

2. **処理の質がシステム全体の成否を決める。**
   HRS完全処理（シナリオ3）は同じ廃棄物から大幅な土壌・植生回復を生み出せる（仮説）。

3. **食料リレーは追加的な相乗効果をもたらす可能性がある。**
   ただし、起動まで約24年かかり（本モデルでの仮説）、
   50年シミュレーション内では完全な成熟には至らない。

4. **残留リスクはゼロにはならない。**
   これはモデルの制約ではなく、現実の衛生リスク管理への警告として意図されている。

5. **水資源（water_stress_factor=0.60）が最大の制約条件。**
   砂漠・乾燥地での実際の適用では、水の調達・管理が最初の課題になる。
