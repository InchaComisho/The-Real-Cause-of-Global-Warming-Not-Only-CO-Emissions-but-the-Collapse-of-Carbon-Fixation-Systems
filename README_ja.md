## 概要

地球温暖化は、一般に二酸化炭素（CO₂）排出量の増加によって説明される。

この説明は、大気物理のレベルでは科学的に正しい。  
CO₂は温室効果ガスであり、放射強制力と地球平均気温の上昇に寄与する。

しかし、この説明が「排出」だけで止まるなら、それは不完全である。

より深い因果層は、地球の自然な炭素固定システムと炭素吸収システムの劣化である。  
そこには、土壌、森林、海洋、湿地、微生物、生物循環ネットワークが含まれる。

つまり、大気中CO₂が増えたのは、人間文明がCO₂を排出したからだけではない。

地球が炭素を吸収し、固定し、貯蔵し、再循環する自然能力そのものが弱体化してきたからでもある。

従来の因果モデルは、次のように表される。

```text
人間活動
→ CO₂排出の増加
→ 地球温暖化
```

本稿では、より深い因果モデルを提案する。

```text
人間活動
→ 土壌・森林・海洋・微生物循環の破壊
→ 炭素固定システムと炭素吸収システムの崩壊
→ CO₂が吸収・固定・再循環されにくくなる
→ 大気中CO₂が蓄積する
→ 地球温暖化が加速する
```

IPCCは、人間の影響が大気、海洋、陸域を温暖化させてきたことは疑う余地がないと述べている。

本稿はその結論を否定しない。

本稿が主張するのは、一般に繰り返される説明では、その前段階にある生態学的原因、つまり炭素循環を制御する自然システムの破壊が十分に語られていない、という点である。

## 中核命題

地球温暖化の本当の原因は、CO₂だけではない。

CO₂は温暖化の直接的な物理要因である。  
しかし、大気中CO₂が継続的に上昇することは、惑星規模の炭素循環が損傷している症状でもある。

核心は、炭素固定システムの崩壊である。

```text
地球は、炭素を過剰に押し込まれているだけではない。
地球は、炭素を処理する能力を失いつつある。
```

したがって、気候対策は脱炭素だけを超えなければならない。

次の段階の気候対策は、以下である。

```text
排出削減から
炭素固定源回復へ。
```

## 1. なぜ従来の説明は不完全なのか

地球温暖化の主流説明は、一般に次のように単純化される。

```text
人間活動がCO₂を排出する。
CO₂が大気中に蓄積する。
温室効果が強まる。
地球が温暖化する。
```

これは間違いではない。

しかし、不完全である。

それはプロセスの後半を説明しているが、完全な因果連鎖を説明していない。

欠けている問いは、次である。

```text
なぜCO₂はこれほど持続的に蓄積しているのか。
```

答えは、単に以下ではない。

```text
人間がCO₂を排出しているから。
```

より深い答えは、以下である。

```text
地球の自然な炭素吸収源と炭素固定システムが劣化しているから。
```

もし土壌、森林、海洋、湿地、微生物、生物循環システムがより健康な状態を保っていれば、大気中の炭素のより大きな部分は吸収、固定、再循環されていた可能性がある。

これは、CO₂排出が無害だという意味ではない。

CO₂蓄積は、原因であると同時に結果でもある、という意味である。

CO₂は大気レベルでは温暖化の原因である。  
同時に、CO₂蓄積は惑星システムレベルでは炭素循環不全の結果でもある。

---

## 2. CO₂は原因であり、症状でもある

CO₂は温室効果ガスである。

大気中CO₂の増加は、地球の放射収支に影響し、温暖化に寄与する。

しかし、CO₂を唯一の根本原因として扱うと、解決策の範囲が狭くなる。

その結果、支配的な答えは一つになる。

```text
排出を減らす。
```

排出削減は必要である。

しかし、それだけでは不十分である。

なぜなら、排出量を減らしても、以下は自動的に回復しないからである。

```text
劣化した土壌は自動的に回復しない。
微生物ネットワークは即座に再構築されない。
森林生態系はすぐに複雑性を取り戻さない。
海洋循環は自動的に安定しない。
湿地や有機炭素貯蔵庫は自然に再出現しない。
```

より深い問題は、どれだけCO₂を排出するかだけではない。

地球がいまも炭素を吸収し、固定する生物的・地質的・生態的能力を持っているかどうかである。

## 3. 炭素固定システムとは何か

炭素固定システムとは、地球システム内で炭素を捕捉し、貯蔵し、変換し、再循環する自然メカニズムである。

それらには、以下が含まれる。

```text
森林
土壌
土壌有機物
腐植
湿地
泥炭地
海洋
植物プランクトン
海洋生物ポンプ
土壌微生物
菌類
根圏微生物ネットワーク
沿岸生態系
草地
農地土壌
```

これらは受動的な景色ではない。

惑星インフラである。

それらは、炭素、水、栄養、温度、生物多様性、生態系安定性を制御している。

土壌だけでも巨大な炭素貯蔵庫である。

土壌劣化は、農業問題だけではない。  
気候問題であり、炭素循環問題であり、文明問題である。

---

## 4. 微生物の見えない役割

微生物は、公共の気候議論ではしばしば無視される。

しかし、微生物は炭素循環の中心にいる。

土壌微生物は、有機物を分解し、栄養を変換し、土壌有機炭素を形成し、植物の根と相互作用し、長期的な炭素貯蔵に寄与する。

近年の研究では、微生物ネクロマス炭素が土壌有機炭素の重要な構成要素であり、長期的な炭素隔離に大きな役割を持つことが示されている。

つまり、微生物は単なる分解者ではない。

炭素処理者である。

炭素がすぐ大気へ戻るのか、土壌中で安定化するのかを左右する生物的機械の一部である。

微生物生態系が崩壊すれば、炭素循環は弱まる。

その結果は、土壌肥沃度の低下だけではない。

惑星規模の炭素固定システムの弱体化である。

## 6. 提案する因果モデル

本稿が提案する因果モデルは、以下である。

```text
人間中心の短期開発
→ 土壌・森林・海洋・湿地の破壊
→ 微生物衰退と生物多様性喪失
→ 炭素固定システムの弱体化
→ 炭素吸収源の弱体化
→ 大気中CO₂の蓄積
→ 地球温暖化
→ 干ばつ・火災・海洋ストレス・土壌劣化
→ 炭素固定システムのさらなる弱体化
```

このモデルでは、地球温暖化はフィードバックループとして理解される。

CO₂蓄積は温暖化を起こす。  
温暖化は炭素吸収源を弱める。  
弱まった炭素吸収源は、より多くのCO₂を大気中に残す。  
この循環が加速する。

したがって、排出だけでなく、吸収源の安定性と炭素固定能力にも焦点を当てる必要がある。

## 7. なぜ脱炭素だけでは解決できないのか

脱炭素は必要である。

しかし、脱炭素だけでは損傷した炭素循環を回復できない。

排出削減は、漏れているシステムへの水の流入を減らすことに似ている。

しかし、貯水池そのものが壊れているなら、流入を減らすだけでは不十分である。

気候システムには、以下の両方が必要である。

```text
1. 排出削減
2. 炭素固定源回復
```

政策が排出だけに集中すると、炭素を吸収し貯蔵する生物的・生態的システムを無視してしまう。

そのため、気候対策には以下が必要である。

```text
土壌再生
森林再生
腐植形成
湿地再生
泥炭地保護
微生物生態系回復
生物多様性回復
海洋生態系回復
海洋栄養循環支援
有機物循環
不要な焼却の削減
再生農業
水循環回復
```

これらがなければ、地球は余剰炭素を処理する能力において、構造的に弱体化したままである。

## 8. なぜジオエンジニアリングだけでは不十分なのか

それらは温度という症状を対象にする可能性がある。

しかし、より深い原因が炭素循環の崩壊であるなら、症状制御だけでは不十分である。

目標は地球システムを支配することではない。

目標は、地球システムが自然に自己調整する能力を回復することである。

本稿では、以下を区別する。

```text
気候症状の人工制御
```

と、

```text
自然な炭素循環の回復
```

である。

より根本的なのは後者である。

目的は自然を置き換えることではない。

自然が再び機能できる条件を修復することである。

大気だけが関係しているわけではない。

関係するシステムには、以下がある。

```text
大気炭素プール

土壌有機炭素プール
海洋炭素プール
生物ポンプ
微生物炭素ポンプ
植生炭素貯蔵

土地利用変化による排出
炭素吸収源効率
```

単純化した技術モデルは以下である。

```text
大気中CO₂増加 =

- 陸上炭素吸収

```

したがって、大気中CO₂は以下の場合に増える。

```text
排出が増える
または炭素吸収源が弱まる
または貯蔵炭素が放出される
または生物固定が低下する
または複数要因が同時に起きる
```

主流の公共的語りは、第一の要因を強調する。

本稿は、方程式全体を強調する。

## 13. 結論

地球温暖化は、CO₂排出だけによって起きているわけではない。

CO₂は温暖化の直接的な物理要因である。  
しかし、その継続的蓄積は、より深い惑星的失敗の症状でもある。

その失敗とは、炭素固定システムと炭素吸収システムの崩壊である。

地球が温暖化しているのは、人間が炭素を排出しているからだけではない。

人間文明が、炭素を吸収し、固定し、貯蔵し、再循環していた自然システムを傷つけてきたからである。

したがって、次の段階の気候対策は以下である。

```text
排出削減
+
炭素固定源回復
+
微生物・生態循環回復
```

気候戦略の未来は、以下へ移行すべきである。

```text
脱炭素だけから
惑星炭素代謝の回復へ。
```

あるいは、より簡潔に言えば、

```text
CO₂を減らすことから
地球がCO₂を処理する能力を回復することへ。
```

## 拡張フレームワーク：炭素吸収源崩壊と再生

炭素吸収源崩壊とは、森林、土壌、海洋、湿地、微生物系が弱まり、CO₂を吸収・固定・貯蔵する能力が低下する状態である。

この状態では、同じ排出量でも大気中CO₂の蓄積が加速しやすくなる。

そのため、気候対策は、排出削減と同時に、以下を進める必要がある。

- 土壌OS再起動
- 腐植と有機物循環の回復
- 森林の多様性回復
- 湿地・泥炭地保護
- 海洋循環と植物プランクトン基盤の回復
- 微生物生態系の回復
- 都市水循環と冷却機能の回復

これは、自然を置き換える技術ではない。

壊れた自然循環を補完し、自然が再び働ける条件を整える自然補完科学の方向である。
